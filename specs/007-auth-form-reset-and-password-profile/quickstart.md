# Quickstart: 인증 폼 상태 초기화 및 프로필 비밀번호 변경 지원

## Setup

```bash
cd backend
npm install
npm run dev
```

```bash
cd frontend
npm install
npm run dev
```

## Manual Verification

### 인증 탭 상태 초기화

1. 로그인 탭에 이메일/비밀번호를 입력한다.
2. 회원가입 탭으로 전환해 이전 로그인 입력값이 남지 않는지 확인한다.
3. 회원가입 탭에서 이름/표시 이름/이메일/비밀번호/확인값을 입력한다.
4. 다시 로그인 탭으로 돌아가 회원가입 입력값과 오류가 남지 않는지 확인한다.

### 비밀번호 변경

1. 로그인 후 프로필 화면으로 이동한다.
2. 현재 비밀번호, 새 비밀번호, 새 비밀번호 확인을 입력한다.
3. 새 비밀번호 확인이 다르면 서버 요청 전 오류가 표시되는지 확인한다.
4. 올바른 현재 비밀번호와 새 비밀번호로 저장 후 성공 메시지와 입력 초기화를 확인한다.
5. 로그아웃 후 새 비밀번호로 다시 로그인 가능한지 확인한다.
